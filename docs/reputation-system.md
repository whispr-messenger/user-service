# Système de réputation

## Score

Chaque utilisateur a un score de réputation.

```
Action positive ──▶ Score +
  (messages, réactions)

Action négative ──▶ Score -
  (signalements, sanctions)
```

## Impact

| Score | Conséquence |
|-------|-------------|
| Élevé | Aucune restriction |
| Moyen | Surveillance |
| Bas | Restrictions (rate limit renforcé) |
